# My-FMCG-Retail-Sales-and-Profitability-Analysis
FMCG Sales & Profitability Dashboard (Excel)
📌 Project Description

This project analyzes FMCG retail sales data to uncover trends in revenue, product performance, and regional sales using Microsoft Excel.
<img width="1603" height="850" alt="image" src="https://github.com/user-attachments/assets/76b0f948-bbdf-4040-80a7-de6fc6543f4e" />

🎯 Objectives
Analyze sales performance across cities and store formats
Identify top-performing brands and categories
Generate insights for business growth
🛠️ Tools Used
Microsoft Excel
Pivot Tables
Charts & Visualizations
Dashboard (KPI Cards + Slicers)
📈 Key Insights
Revenue exceeds INR 39M
Kolkata is the top-performing city
ITC is the leading brand
Hypermarkets generate the highest revenue
Sales peak in August
🚀 Recommendations
Invest in high-performing store formats
Optimize sales strategies during peak months
Focus on top-performing brands and categories
